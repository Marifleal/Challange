Cadastre-se

Esta página irá requerer as informações para cadastro do usuário seja ele candidato ou recrutador.

Os requisitos para cadastro do candidato são:

Nome, endereço, telefone, e-mail, Hard Skills, Soft Skills, idiomas, pretensão Salarial, importar curriculo via PDF, importar currilo de plataformas terceiros.

Os requisitos para cadastro do recrutador são:

Razão Social, CNPJ, Endereço, Contato, Porte da empresa, Descrição da empresa, Setor de atuação, Cadastro de vagas.

Inserir botão para selecionar a opção de CANDIDATO x RECRUTADOR.

Formulário para inserção dos dados após validação de candidato ou recrutador.
